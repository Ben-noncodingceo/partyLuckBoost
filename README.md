# 🎉 派对抽奖系统

一个功能丰富、界面美观的在线抽奖系统，支持批量导入、实时动画、主题切换等功能。

## ✨ 功能特点

### 1. 名单管理
- **Excel 批量上传**：支持 .xlsx 和 .xls 格式，自动读取第一列的所有名字
- **手动管理**：可以手动添加、修改、删除参与者
- **重名检测**：自动检测重名情况并用黄色背景高亮显示
- **容量限制**：支持最多 500 人参与抽奖

### 2. 抽奖配置
- **多轮抽奖**：支持设置 1-10 轮抽奖
- **重复控制**：可选择中奖者是否继续参加后续抽奖
- **灵活设置**：每轮可以自定义奖项名称和获奖人数

### 3. 抽奖动画
- **炫酷滚动**：名字滚动动画，增加紧张感和趣味性
- **时间控制**：最短 1 秒，最长 1 分钟自动停止
- **大按钮操作**：开始/停止按钮切换，操作简单直观

### 4. 结果展示
- **精美卡片**：获奖者以卡片形式展示，带动画效果
- **分轮显示**：清晰展示每一轮的获奖情况
- **结果导出**：可导出为文本文件保存

### 5. 主题背景
- **6 种主题**：默认、圣诞节、新年、生日派对、公司年会、春节
- **自动切换**：根据活动类型自动匹配背景颜色

## 🚀 使用方法

### 方式一：直接打开（推荐）
1. 双击 `index.html` 文件
2. 在浏览器中打开即可使用
3. 无需安装任何软件或配置服务器

### 方式二：本地服务器
```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 然后在浏览器中访问 http://localhost:8000
```

## 📝 使用步骤

### 步骤 1：管理参与者名单
1. **选择活动主题**：在页面顶部选择符合您活动的主题
2. **上传 Excel 文件**：
   - 点击上传区域
   - 选择包含参与者名单的 Excel 文件
   - Excel 文件格式要求：第一列为参与者姓名（可参考 `示例名单.xlsx`）
3. **或手动添加**：
   - 在输入框中输入姓名
   - 点击"添加"按钮
4. **管理名单**：
   - 点击"修改"可以修改姓名
   - 点击"删除"可以删除参与者
   - 重名的参与者会用黄色背景标注
5. **点击"下一步：配置抽奖"**

### 步骤 2：配置抽奖
1. **设置抽奖次数**：输入总共要抽几轮
2. **设置重复规则**：
   - 勾选"允许中奖者参加后续抽奖"：已中奖者可以再次中奖
   - 不勾选：已中奖者不再参与后续抽奖
3. **配置每一轮**：
   - 输入奖项名称（如：特等奖、一等奖等）
   - 输入该轮获奖人数
4. **点击"开始抽奖"**

### 步骤 3：开始抽奖
1. **查看当前轮次信息**：页面显示当前是第几轮，奖项名称和人数
2. **点击"开始"按钮**：名字开始滚动
3. **点击"停止"按钮**：
   - 最少需要滚动 1 秒后才能停止
   - 如果不停止，最长会滚动 1 分钟后自动停止
4. **查看获奖者**：停止后会显示本轮获奖者名单
5. **继续下一轮**：点击确认继续下一轮抽奖

### 步骤 4：查看结果
1. **查看所有获奖者**：按轮次展示所有获奖结果
2. **导出结果**：点击"导出结果"按钮，保存为文本文件
3. **重新开始**：点击"重新开始"可以重新进行抽奖

## 📊 Excel 文件格式

Excel 文件要求：
- 支持 `.xlsx` 和 `.xls` 格式
- 只读取第一列的数据
- 第一列可以有标题行（会自动过滤空行）
- 示例：

| 姓名 |
|------|
| 张三 |
| 李四 |
| 王五 |
| 赵六 |

## 💡 使用建议

1. **重名处理**：如果有重名，建议在名字后加上编号或部门，如"张三-1"、"张三-2"
2. **测试抽奖**：正式抽奖前，建议先用少量名单测试功能
3. **大屏展示**：抽奖时建议使用投影仪或大屏幕，效果更好
4. **保存结果**：抽奖结束后及时导出结果，刷新页面会清空数据
5. **浏览器兼容**：推荐使用 Chrome、Firefox、Edge 等现代浏览器

## 🔧 技术栈

- **HTML5 + CSS3**：响应式设计，支持各种屏幕尺寸
- **JavaScript**：纯 JavaScript 实现，无需框架
- **SheetJS**：用于读取 Excel 文件
- **CSS 动画**：流畅的过渡和动画效果

## ⚠️ 注意事项

1. **人数限制**：系统限制最多 500 人参与抽奖
2. **时间控制**：抽奖滚动时间最短 1 秒，最长 1 分钟
3. **数据保存**：数据仅保存在浏览器中，刷新页面会清空，请及时导出结果
4. **网络要求**：首次使用需要联网加载 SheetJS 库，之后可离线使用

## 🎯 常见问题

**Q: Excel 文件上传后没反应？**
A: 请确保 Excel 文件格式正确，第一列包含有效的姓名数据。

**Q: 可以同时抽取多个人吗？**
A: 可以，在配置抽奖时设置"获奖人数"即可。

**Q: 如何让同一个人可以多次中奖？**
A: 在配置页面勾选"允许中奖者参加后续抽奖"。

**Q: 可以修改已经导入的名单吗？**
A: 可以，在名单管理页面可以添加、修改、删除参与者。

**Q: 导出的结果是什么格式？**
A: 导出为 `.txt` 文本文件，包含所有轮次的获奖者名单。
